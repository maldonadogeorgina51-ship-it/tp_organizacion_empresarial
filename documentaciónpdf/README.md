# Proyecto: Análisis de Ventas
## Integrantes: Hugo, Paco y Luis.

## Descripción
Este proyecto automatiza el procesamiento de datos de ventas mediante un script de Python. El sistema lee un archivo CSV, realiza cálculos de ventas totales, agrupa por mes y genera un reporte visual.

## Estructura del Proyecto
- /datospdf: Contiene sales_data.csv (Datos fuente).
- /scriptspdf: Contiene analisis_datos.py (Lógica del script).
- /resultadospdf: Contiene metricas_finales.txt y grafico.png (Resultados).

## Instrucciones de Ejecución
1. Asegurarse de tener instalado pandas y matplotlib.
2. Ejecutar analisis_datos.py.
3. Los resultados aparecerán en la carpeta /resultadospdf.
